# S.N.O.O.P
> System Navigation & Overhead Observation Panel

---

## Overview

S.N.O.O.P was developed to give users more insight into how their RAM grows or decays over a given scale of time, small or big. The program allows you to take a number of snapshots in a given timeframe of RAM usage, and provides a table for easy viewing.

### Tech Stack

- Java
- JavaFX
- Maven
- OSHI

### Prerequisites


- Java 21.0.11
- Maven 3.8.7


## Installation & Running
Open the terminal and run these commands:
```bash
git clone https://github.com/azzam-c/snoop.git
cd snoop
```
To run the program, run this command:
```bash
mvn clean javafx:run
```
